# OIBSIP Web Development Internship

## Level 2 – Task 3: Basic To-Do Web App

### Objective
The objective of this project is to create a simple To-Do List web application where users can add and manage their daily tasks.

### Technologies Used
- HTML
- CSS
- JavaScript

### Features
- Add new tasks
- Display list of tasks
- Mark tasks as completed
- Delete tasks from the list
- Simple and user-friendly interface

### Project Structure
todo-webapp/
├── index.html
└── README.md

### Outcome
This project demonstrates the use of basic web technologies to build an interactive To-Do List application that helps users organize tasks efficiently.
